# 圖片資源說明

## 小助手頭像設定

請將你想要使用的小助手頭像圖片放在此資料夾，並命名為：

- **assistant-avatar.png** - 小助手頭像（推薦）
- **assistant-avatar.jpg** - 小助手頭像（備選）
- **assistant-avatar.gif** - 小助手頭像（支援動畫）

**建議規格：**
- 尺寸：128x128 像素或更大（會自動縮放）
- 格式：PNG（支援透明背景）、JPG、GIF
- 檔案大小：< 500KB

**使用方式：**
1. 將你的圖片複製到此資料夾
2. 重新命名為 `assistant-avatar.png`（或 .jpg / .gif）
3. 重新整理瀏覽器（Ctrl+Shift+R）即可看到新頭像

**注意事項：**
- 圖片會自動裁切為圓形
- 如果找不到自訂圖片，會顯示預設 emoji 😊
- 支援透明背景（推薦使用 PNG 格式）
