# 🖼️ 如何更換使用者頭像

## 📝 快速指南

### 步驟 1：準備你的圖片
1. 找一張你想用的圖片（建議使用圓形或正方形）
2. 調整尺寸（建議 128x128 像素或更大）
3. 儲存為 PNG、JPG 或 GIF 格式

### 步驟 2：放置圖片
將圖片複製到以下資料夾：
```
static/images/user_avatars/
```

### 步驟 3：重新命名
將圖片重新命名為以下格式：
- `user-avatar.png` （推薦，支援透明背景）
- `user-avatar.jpg`
- `user-avatar.gif` （支援動畫）

### 步驟 4：重新整理
在瀏覽器按 **Ctrl+Shift+R** 強制重新整理頁面

## ✨ 建議規格

**圖片尺寸：**
- 最小：64x64 像素
- 建議：128x128 或 256x256 像素
- 最大：512x512 像素（避免檔案過大）

**圖片格式：**
- **PNG** - 最推薦（支援透明背景）
- **JPG** - 適合照片
- **GIF** - 支援動畫效果

**檔案大小：**
- 建議小於 500KB
- 確保載入速度流暢
