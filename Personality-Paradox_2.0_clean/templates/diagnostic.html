常見問題排解

❌ ChatBot 顯示「無法連接」錯誤

原因：Ollama 服務未啟動

解決方法：

  1. 開啟終端機（PowerShell 或 CMD）
  2. 執行命令：ollama serve
  3. 保持終端機開啟
  4. 重新整理 ChatBot 頁面
❌ 模型未找到錯誤

原因：gemma3:4b 模型未下載

解決方法：

  1. 開啟終端機
  2. 執行命令：ollama pull gemma3:4b
  3. 等待下載完成（約 2-3 GB）
  4. 再次嘗試使用 ChatBot
📋 快速啟動指令

1. 啟動 Ollama：

ollama serve

2. 下載模型（首次使用）：

ollama pull gemma3:4b

3. 啟動 Flask：

python app_flask.py
